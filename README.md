## 📂 Proyectos Destacados

### ☀️ [Análisis de Viabilidad Solar](https://github.com/idk-Damian)
Estudio estadístico utilizando conjuntos de datos de **NASA POWER** (2020-2025) para calcular la varianza de irradiancia en Ambato. Se aplicaron pruebas de hipótesis (Z-tests).

### ⚙️ [Generador de Código EMF](https://github.com/idk-Damian)
Herramienta de automatización de código *full-stack* que utiliza **Ecore** y **Xtend** para crear componentes funcionales de interfaz de usuario y esquemas de base de datos a partir de modelos de alto nivel.
